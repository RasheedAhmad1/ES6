JavaScript Functions

A JavaScript function is a block of code designed to perform a particular task.
A JavaScript function is executed when "something" invokes it (calls it).

A simple function

Use the keyword 'function' to declare a function.
Use 'function' name with parenthesis to invoke it.

function printName() {
console.log('Rasheed Ahmad');
}
printName();
 Output: Rasheed Ahmad

function with parameters

Use variable name(s) in the parenthesis as parameters
and then pass values when you call a function as arguments.

function addNumbers(a, b) {
var sum = a + b;
console.log(sum);
}
printName(2, 3);
 Output: 5

function returning a value

Functions can also return a value, use the return keyword to return something.
To use a return keyword, we have to store it in a variable to use it then.

var result;
function multiplyNumbers(a, b) {
var multiply = a * b;
return multiply; }
result = multiplyNumbers(2, 3);
console.log(result);
 Output: 6

Scope and Environment

Scope is the area where a variable can be used.
There are global scope and local scope

var num1 = 5; // global scope - can be used anywhere in the program
function sum (a) {
var num2 = 2; // local to this function - only this function can use it
var result = num1 + num2 + a;
console.log(result);
}
sum(5);
 Output: 12

num1 is a global variable thus the function can use it,
but num2 cannot be used outsite the function.
