Javascript Variables

Variables Naming Conventions

  • A variable name can containg letters, digits, underscre and a dollar sign
  • Can not start with number
  • But can start with an underscore or dollar sign
  • Case sensitive
  • Can not be reserved words

JS variables can be defined by using var, let, or const keywords

Using var

  • var numOne = 10;
  • var numTwo = 5;
  • var sum = numOne + numTwo;
  • console.lgo(sum);

Using let

  • let numOne = 10;
  • let numTwo = 5;
  • let sum = numOne + numTwo;
  • console.lgo(sum);

Using const

  • const numOne = 10;
  • const numTwo = 5;
  • const sum = numOne + numTwo;
  • console.lgo(sum);

Naming notations for JS

Camel Case:
Combine multiple words but first letter of the words proceding the
first word will be capital i.e firstName
Better to use for variables and functions name
Pascal Case:
Combine multiple words but first letter of all the words will be capital i.e FirstName
Snake Case:
Combine multiple words with an underscore i.e first_name
Better to use for files names
