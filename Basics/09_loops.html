JavaScript Loops

The repetition of code is called Loop and each repetition is called iteration.
In another words, running the same lines of code for a certain number of times is c/d Loop.
JavaScript offers many types of Loops - while loop, for loop and, forEach loop.

while() Loop

When you have to stop at a certain condition is reached, you can use while Loop then.

var x = 1;
while(x <= 3) {
console.log(x);
x++;
}
 Output:
1
2
3

While the condition is true always execute the code block inside loop

for() Loop

When you know, how many times the Loop need to iterate then you should use for Loop.

var numbers = [1, 2, 3];
var pos;
for(pos = 0; pos<numbers.length; pos++) {
console.log('Position => ' +pos+ ' Value => ' + numbers[pos]);
 Output:
Position => 0 Value => 1
Position => 1 Value => 2
Position => 2 Value => 3
Position => 3 Value => 4

Thus the loop iterates upto the length of the array

break keywork

Breaking for loop

var numbers = [1, 2, 3, 4, 5];
var pos;
for(pos = 0; pos < numbers.length; pos++) {
if(pos == 2) break;
console.log('Position => ' +pos+ ' Value => ' + numbers[pos]);
}
 Output:
Position => 0 Value => 1
Position => 1 Value => 2

Thus 'break' terminates the loop where the given 'if' condition is met.

continue keywork

Skip a certain iteration to start the next iteration.

Skipping for loop, printing even numbers.

var numbers = [1, 2, 3, 4, 5, 6];
var pos;
for(pos = 0; pos < numbers.length; pos++) {
if(pos%2==0) continue;
console.log(numbers[pos]);
}
 Output:
2
4
6

Thus the 'continue' keyword skips the iteration where the given 'if' condition is mets.
