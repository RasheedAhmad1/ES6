JavaScript Loops

The repetition of code is called Loop and each repetition is called iteration.
In another words, running the same lines of code for a certain number of times is c/d Loop.

JavaScript offers two types of Loops - While Loop and for Loop

When you know, how many times the Loop need to be iterate then you can use for Loop.

When you have to stop at a certain condition is reached, you can use while Loop.

         
While Loop

         
            //While the condition is true always execute the code block inside loop
            var x = 1;
            while(x<=10) {
              console.log(x);
              x++;
            }
         
         
For Loop

         
            //for loop
            var numbers = [1, 2, 3, 4, 5];
            var pos;
            for(pos = 0; pos ' +pos+ ' Value => ' + numbers[pos]);
            }
