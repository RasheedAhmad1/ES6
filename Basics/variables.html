// Variables // // Variables Naming conventions: // - A variable can contain letters, digits, underscore and dollar sign // - Can not start with a number // - But can start with a an underscore or dollar sign // - Case sensitive // - Can not be reserved words var numOne = 10; var numTwo = 5; var sum = numOne + numTwo; alert(sum); // Operators // // Operators are symbols used to perform some actions on variables. In the above exmple "+" is an operator used to add two values. // Program flow // // Javascript code executes from top to bottom. It executes one line of code at a time, if an error occurs then the next line of code is not executed. // Expression // // Ecpression is a line of code which contains a valid combination of variables, values, and operators e.g: (num1+num2)/sum; // Naming notations for Js // // Camel Case (Upper Camel Case): firstName; for variables and functions names // Pascal Case(Lower Camel Case): FirstName; // Snake Case (Underscore): first_name; for files name
