JavaScript Arrays

Arrays are special type of object which stores a list of data

var person = ['Rasheed Ahmad', 26, 'Peshawar']
console.log(person);
 Output: ['Rasheed Ahmad', 26, 'Peshawar']

Arrays are zero indexed, means that the first element resides at person[0] and so on

person[0] --> 'Rasheed Ahmad'
person[1] --> 26
person[2] --> 'Peshawar'

Check the type of the array

console.log(typeof(person));
 Output: object

Update an existing value say 'second value'

person[1] = 25;
console.log(person[1]);
 Output: 25

Add new value at the end of the Array

person[person.length] = 'Engineer';
console.log(person);
 Output: ['Rasheed Ahmad', 25, 'Peshawar', 'Engineer']

OR adding with a built-in function

person.push('Software');
console.log(person);
 Output: ['Rasheed Ahmad', 25, 'Peshawar', 'Engineer', 'Software']

Remove last element of the Array

person.pop();
console.log(person);
 Output: ['Rasheed Ahmad', 25, 'Peshawar', 'Engineer']

Add or Remove a value at a specific location

person.splice(2, 1, 'Nice Person');
console.log(person);
 Output: ['Rasheed Ahmad', 25, 'Nice Person', 'Engineer']

Parameters:
- First - starting position
- Second - how many items to be deleted
- Third - add element

Delete a specific element

person.splice(2, 1);
console.log(person);
 Output: ['Rasheed Ahmad', 25, 'Engineer']

Parameters:
- First - starting position
- Second - how many items to be deleted

Check length of the array

console.log(person.length);
 Output: 5

Concatination: Concatinating two arrays

Languages = ['JavaScript', 'React', 'Node.js'];
var newData = person.concat(Languages);
console.log(newData);
 Output: ['Rasheed Ahmad', 25, 'Engineer', 'JavaScript', 'React', 'Node.js']

Sorting Ascending and Descending

newData.sort(); // Ascending order
newData.reverse(); // Descending order
 Output:
[25, 'Engineer', 'JavaScript', 'Node.js', 'Rasheed Ahmad', 'React']
['React', 'Rasheed Ahmad', 'Node.js', 'JavaScript', 'Engineer', 25]

Some Useful Array Methods

const items = [
{ name: 'Bike', price: 200 },
{ name: 'Phone', price: 150 },
{ name: 'TV', price: 202 },
{ name: 'Movie', price: 10 },
{ name: 'Cycle', price: 140 },
{ name: 'Computer', price: 180 },
{ name: 'Mouse', price: 20 }
]

filter(): Filetering Data

Fileter those items whose price is less than or equal to 100 dollars

const filterdItems = items.filter( (item) => {
return item.price <= 100;
});
console.log(filterdItems);
 Output: [
{name: 'Movie', price: 10},
{name: 'Mouse', price: 20}
]

map(): Mapping Data

Convert the array into a new array

const mapedItems = items.map( (item) => {
return item.name;
});
console.log(mapedItems);
 Output: ['Bike', 'Phone', 'TV', 'Movie', 'Cycle', 'Computer', 'Mouse']

find(): Finding an element

Find the 'Cycle' in the array

const foundItem = items.find( (item) => {
return item.name === 'Cycle';
});
console.log(foundItem);
 Output: {name: 'Cycle', price: 140}

some(): Check if anything in an array returns true of false

For example if the array has inexpensive items return true else return false

const inexpensiveItems = items.some( (item) => {
return item.price <= 50;
});
console.log(inexpensiveItems);
 Output: true

Thus the array has atleast one item less than or equal to 50 dollars

every(): some() checks atleast one item false under the condition

But every() checks each and every item should fall under the condition.

const everyItem = items.every( (item) => {
return item.price <= 50;
return item.price <= 202;
});
console.log(everyItem);
 Output: true

Returns 'false', because there are some items greater than 50 dollars

The second return statement returns 'true', because every item is less than or equal to 202 dollars
