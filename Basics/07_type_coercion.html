Type Coercion

Conversion of data type from one to another is called Type Coercion

Coercion have two types

  • Imlicit Coercion: Run automatically by JavaScript
  • Explicit Coercion: Done programatically by us

Imlicit Coercion Example:

Example

Click the button to see the type of the result.

var numOne = 5;
var numTwo = "Hello";
var sum = numOne + numTwo;

Output:

Type:

Explanation:

Code

HTML Code

<p id="trial">Click the button to see the type of the result.</p>
<button type="button" onclick="loadCoercion()">
<span class="button-text">Check Type!</span>
</button>

JavaScript Code

<script>
var numOne = 5;
var numTwo = "Hello";
var sum = numOne + numTwo;
const info = "The result is 'String', because JS adds a number and string and the result is a string also";
function loadCoercion() { document.getElementById('output').innerHTML = sum; document.getElementById('type').innerHTML = typeof(sum); document.getElementById('exp').innerHTML = info; }
<p>Output:</p><samp id="output"></samp>
<p>Type:</p><p id="type"></p>
<p>Explanation:</p><p id="exp"></p>
</script>
