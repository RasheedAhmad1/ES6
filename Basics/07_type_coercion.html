Type Coercion

Conversion of data type from one to another is called Type Coercion

Have two types

  • Imlicit Coercion: Run automatically by JavaScript
  • Explicit Coercion: Done programatically by us

Imlicit Example:

            
                var numOne = 5;
                var numTwo = "Hello";
                var sum = numOne + numTwo;
                console.log(sum);
            
            
                "Hello5"
                The result is a string which is automatically done by JS depending upon the operation
                Check its type - typeof(sum) - string

Explicit Example:

  • To Number: Number()
  • To String: String()
  • To Boolean: Boolean()
            
                
                console.log(Number(true), typeof(Number(true))); --> 1 Number 
                console.log(Number(5), typeof(Number(5))); --> 5 Number
                console.log(Number(-2), typeof(Number(-2))); --> -2 Number
                console.log(Number(2.5), typeof(Number(2.5))); --> 2.5 Number 
                console.log(Number('4'), typeof(Number('4'))); --> 4 Number
                console.log(Number('Hello'), typeof(Number('Hello'))); --> NaN Number
