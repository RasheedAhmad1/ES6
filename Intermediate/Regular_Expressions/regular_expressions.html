Regular Expressions

A regular expression is an object that describes a pattern of charctors.
It allows us to search for specific patterns of text using the difined pattern.

    For example:
  • To check whether a given string is a correct email address or not
  • To check that an entered username only contains alphabets and numbers or not

Syntax

/pattern-string/flags
pattern-string is where you define what strings this pattern should match.
forward slashes represents the start and end of pattern-string.
flags are used to difine whether you want to search to be case-sensitive or case-insensitive,
or whether you want just the first result, or all the results.

Examples:

  • /something/ -- findout something word, by default reg. exp. returns the first mathched result.
  • /something/g -- if there are multiple words, then return all words. 'g' means global (all)
  • /something/i -- if there are all caps, then return all. 'i' means case-insensitive.
  • /something/gi -- return case-insensitive and global results.

Sets and Ranges

Sets and ranges are represented by several charactors inside square brackets [...].
This means that findout any charactor among given values.
It has two variations:
/something/ -- find something in the String /something/g -- by default JS returns the first word matches, but if we want to return multiple strings the use 'g' flag for globalThis /something/i -- 'i' for case-insensitive, to return all caps [ABC] - It means any of the given three chartors, it is a set.
Example:
String: Air Brim Circle
Pattern: [ABC]irm
Output: Birm

Squar bracktes may also contain charactors range.
[A-Z] is a charactor in range from A to Z.
[0-9] is a digit from 0 to 9.
Example:
String: Same Name Fame Glam
Pattern: [A-Z]ame
Output: Same Name Fame

Charactor Class

\d ([0-9]): select all numbers.
\D (negation of samll 'd') all charactors other than numbers.
^ : (carrot/cap sign) other than -- [^0-9] --> will select all charators other than these numbers.
\w ([a-zA-Z0-9]): all charactors (allcaps) and numbers.
\W (negation of small 'w'): select all special charactors and spaces.
\s : select all the spaces.
\S (negation of small 's'): select everythin except spaces.

Regex Quantifiers

Sets and Ranges selects a value single time, but 'Quantifiers' are selecting a value
multiple times.
Syntax: Add a value in curly braces after sets and ranges.
All numbers with three digits only
Example:
String: He is born in 1996 and now 26 years old.
Pattern: [0-9]{2}
Output: 19 96 26
Can use a range of quantifiers also
[0-9]{2, 3} --> select from two to four numbers
Output: 1996 28
{0, } = * --> zero and onwards
{0, 1} = ? --> either zero or 1
. ([.]): Everything
+ ([\d]+): Either 1 or more --> Return all numbers proceding all. Without '+' it will return an array of numbers but with '+' it returns a string.

Tasks: Select only the hased color codes.
String: color: #123456, background-color: #AB12C8, forground-color: #ADEFCB
Pattern: #[a-zA-Z0-9]{6} or #[\w]{6}
^(\+92)?[\d]{7}$: 92 is optional here
URL pattern: /www\.[\w]+\.(com|co|org|in)$/g

Start and End conditions

^ : start (put at the begging)
$ : end with
Example1: ^#[\w]{6}$ - #ADEFCB --> matched
Example2: ^#[\w]{6}$ - #ADEFCB1234 --> no match

Regular Expressions using JS built-in functions

  1. str.search(pattern): Returns the position of the first match.
  2. pattern.test(str): Returns true if there is a match and false if no match.
  3. str.match(pattern): Returns the matched string. without 'g' returns a single match and with 'g' returns multiple matches.
