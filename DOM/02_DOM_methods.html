JavaScript - HTML DOM Methods

HTML DOM methods are actions you can perform (on HTML Elements).
HTML DOM properties are values (of HTML Elements) that you can set or change.

The DOM Programming Interface

The HTML DOM can be accessed with JavaScript (and with other programming languages).
In the DOM, all HTML elements are defined as objects.
The programming interface is the properties and methods of each object.
A property is a value that you can get or set (like changing the content of an HTML element).
A method is an action you can do (like add or deleting an HTML element).

Example:

The following example changes the content (<font color="#ff0000">the innerHTML</font>) of the <font color="#ff0000">

</font> element with <font color="#ff0000">id="demo"</font>:

        
             <html>
              <body>
                <p id="demo"></p>
                    
                <script>
                 document.getElementById("demo").innerHTML = "Hello World!";
                </script>
                
              </body>
             </html>

In the example above, <font color="#ff0000">getElementById</font> is a method,
while <font color="#ff0000">innerHTML</font> is a property.

The getElementById Method

The most common way to access an HTML element is to use the <font color="#ff0000">id</font> of the element.
In the example above the <font color="#ff0000">getElementById</font> method used <font color="#ff0000">id="demo"</font> to find the element.

The innerHTML Property

The easiest way to get the content of an element is by using the <font color="#ff0000">innerHTML</font> property. The <font color="#ff0000">innerHTML</font> property is useful for getting or replacing the content of HTML elements.

The <font color="#ff0000">innerHTML</font> property can be used to get or change any HTML element, including <font color="#ff0000"><html></font> and <font color="#ff0000"><body></font>.

The HTML DOM Document Object

The document object represents your web page.
If you want to access any element in an HTML page, you always start with accessing the document object.
