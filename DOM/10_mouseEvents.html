Mouse Events

The mouse events are triggered when the different keys of mouse are pressed and/or released.

They are many

mousedown:
Mouse down event is triggered when either the left or right (or middle) mouse key is pressed on an HTML element.
mouseup:
Mouse up event is triggered when either the left or right (or middle) mouse key is relaesed.
click:
Click event is triggered when the left mouse button is pressed and released on the same HTML element, it requires the mousedown and mouseup event before click event.
dbclick:
Double click event is triggered when the mouse butthon is clicked twice.
mouseover:
Mouse over event is triggered when the mouse pointer enters the HTML element, and its child elements.
mouseenter
Mouse enter event is triggered when the mouse pointer enters the HTML element.
mousemove
Mouse move envent is triggered when every time the mouse is moved over the HTML element.
