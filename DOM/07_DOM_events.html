JavaScript Events

When some action happens in the webpage, that action is called an Event.

A JavaScript can be executed when an event occurs, like when a user clicks on an HTML element.

  • The user clicking mouse over a certain element
  • Hovering the cursor over a certain element
  • Pressing a key on the keyboard
  • Resizing or closing the browsers window
  • A webpage finishing loading
  • A form being submitted
  • A video being played, or paused, or finishing play

Handling events in JavaScript

There are two ways to handle a JavaScript event

  • Using Events Properties
    • onload
    • onunload
    • onchange
    • onmouseover
    • onclick
    • onmousedown
    • onmouseup
  • Using addEventListener()
  • And/OR removeEventListener()
    • resize
    • click
    • mousemove
    • mousedown

Method 1: Using Events Properties

An event listener listens for an event, when a certain event is fired

Syntax: document.getElementById('btn').{eventProperty} = function() {}

Example: document.getElementById('btn').onClick = function() {}

            
Sample Code:

            
                var result = document.getElementById('btn1');
                result.onclick = function () {
                alert('I\'m Clicked');
                }

Method 2: Using addEventListener()

An event listener listens for an event, when a certain eventName is fired

Syntax: document.getElementById('btn').addEventListener({eventName}, function() {});

Example: document.getElementById('btn').addEventListener('click', = function() {})

            
Sample Code:

            
                var result2 = document.getElementById('btn2');
                result2.addEventListener('mouseover', onHoverFunction);

                function onHoverFunction () {
                alert('I\'m hovered!');
                }

Only use the function reference 'function_name' and don't call the function 'function_name()'.

Note that you don't use the "on" prefix for the event; use "click" instead of "onclick".
