JavaScript HTML DOM - Changing CSS

The HTML DOM allows JavaScript to change the style of HTML elements.

Changing HTML Style

There are two ways we can get Element style value:

  1. Inline style
  2. CSS File

Method 1: Inline Style

Syntax: document.getElementById(id).style.{inline-style_property} = new style

The following example changes the style of a <p> element:

            
                <html>
                    <body>
                        <p id="p2">Hello World!</p>
                        <script>
                                document.getElementById("p2").style.color = "blue";
                        </script>
                    </body>
                </html>

Method 2: CSS File

Syntax: window.getComputedStyle('selectedElement').{style_property_name};

Example: window.getComputedStyle('id').marginRight;

Using Events

The HTML DOM allows you to execute code when an event occurs.
Events are generated by the browser when "things happen" to HTML elements:

  • An element is clicked on
  • The page has loaded
  • Input fields are changed

You will learn more about events in the next chapter of this tutorial.
This example changes the style of the HTML element with id="id1", when the user clicks a button:

            
                <!DOCTYPE html>
                <html>
                    <body>
                    <h1 id="id1">My Heading 1</h1>
                    <button type="button" 
                    onclick="document.getElementById('id1').style.color = 'red'">
                    Click Me!</button>
                    </body>
                </html>

Adding or Removing CSS Classes

The modren browsers give us access to a property called "classList",
this classList holds all the classes of an HTML element.

Syntax: selectedElement.classList.add('class_name');
