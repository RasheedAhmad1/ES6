Template Strings

Template string is a string which allows embedding expressions inside it.
To create a template string you don't have to use sigle or double quotes but
instead use back ticks. The experssion is then embedded iside curly braces
following the dollar sign.

Example: const mGreetings = `Hello ${name}` -- Here the expression is a variable
const moreGreetings = `Hello ${fullName()}` -- Here the expression is a function call
